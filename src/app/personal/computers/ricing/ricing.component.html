*NIX Ricing

Since I spend much of my time on computers, I always see fit to customize the environment on them, and I go to pretty far lengths to do so! In the *NIX world this is often called "ricing", inspired by (of all things!) the car customization world.

My ricing toolkit fundamentally consists of:

  • Conky
  • Polybar
  • BSPWM

…and others, like my app launcher, my various keyboard shortcuts & macros, a few scripts, and so on. Everything here can be found at my "dotfiles" git repository, which stores my configuration files and scripts, with different branches for each machine I've riced. These are provided as an educational and reference tool more than a real set of software for use, so be warned: I do not provide support for them!

Modern-Day: Sathanas "World" Style

This is my current desktop, with "dials" for different system stat displays and a nice world-map background.

This is for my desktop, which has dual monitors; this is the right screen view, the left screen is empty except for the bar at the top. All buttons are clickable, VLC controls at the top work due to a hacky telnet script, and the dials and clock update in real time.

However, it didn't start out this way…

 The Original: Ubuntu

Yup, nothing special. Just Ubuntu with a few tweaks applied, from ~2015. It didn't stay like this for long, though — I was soon onto working with Conky and i3!

 Next-Gen: Lazarus

For the longest time, this was my stat panel! Fixed on the top right of my display and with a simple system bar on the bottom, this is what greeted me whenever I started my computer up. Not much to look at, but I suppose it got the job done.

 The Transitionary: IExploder Battlemode

As a small project, I added a second mode to this thing, which would completely change the Conky config file and display a "battlemode" animation. Those laptop outlines on the left are of my actual laptop at the time, with the labels corresponding to where the real hardware on the machine was.

 The Polybar/Freespace Era: Moloch

This is when I finally transitioned to Polybar with completely original config and the same for Conky. I standardized on naming my devices after ships and fighters from the Freespace 2 universe. This is a screenshot of the setup on my laptop, "Moloch", so-named for the Moloch-class corvette in the game. My desktop computer, "Sathanas", is similarly named after the Sathanas-class juggernaut.
