About me

I'm currently attending Worcester Polytechnic Institute (WPI) in Worcester, MA, with a major in computer science and a minor in data science. Although I'm working on a CS degree, I started learning programming and general advanced computer skills on my own in high school, starting with C++. Since then I've been tinkering about with just about everything I can get my hands on, from systems programming and networking to data visualization and web development. In recent years I've branched out into more artistic works, including some of the above 3D modeling work and fancily-typeset documents in LaTeX.

Current Activities

For my senior year capstone I'm working on a data collection & analysis project concerning internet connectivity in the US. It involves several terabytes of data and some fun (read: complicated) data analyses, so I've got more than enough to work with for now!

When I'm not working on that, I'm usually busy working on this site or on web development work for WPI's "Beecology" project. In my down time I've once again become addicted to the game Factorio (check out the games page!), hoping to build some megafactory. Starting I'll be starting a full time job at Amazon Robotics as a software development engineer, so that'll replace all the time I put into school.
